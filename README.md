# AI Automation Portfolio

This repository showcases demo automation scenarios focused on handling and processing customer inquiries using AI.

## Overview
I specialize in designing automation systems that help businesses manage incoming inquiries, reduce response time, and bring structure to customer communication using AI-assisted workflows.

## Demo Case 1: Inquiry Intake & AI Response System
An automation system that ensures no customer inquiry is lost and provides instant AI-powered responses.

**Key problems addressed:**
- Lost messages in messengers
- Slow response times
- Lack of inquiry tracking

**Workflow overview:**
1. Customer sends a message via messenger
2. Inquiry is automatically recorded
3. AI analyzes the message and generates a response
4. A manager is notified when human input is required

**Result for business:**
- Increased response speed
- Higher inquiry conversion
- Full visibility of incoming requests

## Demo Case 2: AI-Based Inquiry Classification & Prioritization
A backend automation system that helps teams process large volumes of inquiries efficiently.

**Key problems addressed:**
- Mixed inquiry types
- Missed urgent requests
- Manager overload

**Workflow overview:**
1. AI analyzes incoming inquiries
2. Classifies inquiry type and priority
3. Suggests response drafts
4. Supports managers in decision-making

**Result for business:**
- Improved processing efficiency
- Reduced manual workload
- Scalable inquiry handling

## Technology Stack
- n8n (automation orchestration)
- Telegram, email, Instagram (communication channel)
- Google Sheets (lead storage)
- AI API (message analysis and response)

## Business Requirements
To implement such systems, the business provides:
- Access to communication channels
- Subscription to an AI service
- Confirmation of response logic and escalation rules
